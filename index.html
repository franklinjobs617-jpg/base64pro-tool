Code Snippets

Python Example:

          
import base64
# Encode
text = "Hello, Base64Pro!"
encoded = base64.b64encode(text.encode())
print(encoded.decode())  # SGVsbG8sIEJhc2U2NFBybw==

# Decode
decoded = base64.b64decode(encoded).decode()
print(decoded)  # Hello, Base64Pro!

JavaScript Example:

          
// Encode
const text = "Hello, Base64Pro!";
const encoded = btoa(text);
console.log(encoded);  // SGVsbG8sIEJhc2U2NFBybw==

// Decode
const decoded = atob(encoded);
console.log(decoded);  // Hello, Base64Pro!
